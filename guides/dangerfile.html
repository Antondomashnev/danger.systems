The Dangerfilea project from orta therox and felix krause.

A Dangerfile is a Ruby DSL. Before the ruby code inside your Dangerfile is executed, she grabs useful bits of data about: the CI environment, the git diff, and the code review details. There is a full writeup of what happens in “What does Danger do?”. For now that’s enough.

The Dangerfile is where you create your rules, there are no hard and things to recommend here I’m afraid.
